```

├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │

│  ─────────────────────                                                              │

│  ┌────────────────────┐                                                            │

│  Step 2: Success State                                                              │
│  ┌────────────────────┐                                                            │
│  │                    │                                                            │
│  │  [✓ Ditambahkan]   │  → Background changes to success green                    │
│  │                    │  → Icon changes from + to ✓                                │
│  │                    │  → Text changes                                            │
│  └────────────────────┘  → Duration: 1.5s, then revert                             │
│                                                                                     │
│  Step 3: Cart Icon Animation (in header)                                            │
│  ┌──────────────────────────────────────────────────────────────────────┐          │
│  │                                                    ┌────┐            │          │
│  │                                                    │ 🛒 │            │          │
│  │                                              ┌───┐ │(+1)│            │          │
│  │                                              │ 2 │◀──── │            │          │
│  │                                              └───┘ └────┘            │          │
│  └──────────────────────────────────────────────────────────────────────┘          │
│                                                                                     │
│  Cart icon: "shake" animation (rotate -10deg → 10deg → 0)                          │
│  Badge: scale(0) → scale(1.2) → scale(1) "pop" effect                              │
│  Badge number: increment animation                                                  │
│                                                                                     │
│  Step 4: (Optional) Flying Product Animation                                        │
│  ┌────────────────────────────────────────────────────────────────────┐            │
│  │                                                          [Cart]    │            │
│  │                                                      ↗             │            │
│  │  ┌──────────────┐                              ↗                   │            │
│  │  │   Product    │  ──────────────────────  ↗                       │            │
│  │  │   Thumbnail  │       (flying)                                   │            │
│  │  └──────────────┘                                                  │            │
│  └────────────────────────────────────────────────────────────────────┘            │
│                                                                                     │
│  - Clone product image as absolute positioned element                               │
│  - Animate position from product card to cart icon                                  │
│  - Scale down during flight                                                         │
│  - Fade out at destination                                                          │
│  - Duration: 0.6s, easing: cubic-bezier(0.4, 0, 0.2, 1)                             │
│                                                                                     │
```
